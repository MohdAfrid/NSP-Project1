FEATURES
1. Music Player
2. Music Streaming
3. JAM Search
4. Mood Cards
5. Ringtone Maker
DESCRIPTION
  • JAM is a music player application based on android and will further available for IOS and web version.
  • It works in both online & offline mode.
ONLINE MODE :
1. User can stream through dozens of song's available.
2. User can select a mood card to listen songs as per their mood.
3. JAM SEARCH - It is a Music Recognition System.
User can easily search an unknown song simply by allowing device's microphone to listen the music playing in surrounding and it will show the search results , either the song is available on Youtube or Web.
OFFLINE MODE :
1. User can play songs available on device's storage.
2. User can change (either increase or decrease) a particular songs speed as well as pitch.(Isn't is fun!)
3. User can simply create ringtone of any song.(No need to use a seperate app for this)
